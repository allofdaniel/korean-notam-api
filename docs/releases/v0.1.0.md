# v0.1.0

Initial public open-source release of Korean NOTAM API and Crawler.

## Highlights

- direct collection path for Korean NOTAM data
- optional Selenium fallback for brittle upstream flows
- normalized JSON output for downstream automation
- SQLite-oriented persistence and change detection building blocks
- self-hostable FastAPI reference API
- sample fixture and tests for local integration work
- English, Korean, Japanese, and Simplified Chinese README entry points

## Why It Matters

Korea does not currently offer an easy public developer-facing NOTAM API. This repository is intended to reduce repeated scraping and normalization work for teams building aviation research tools, flight-planning prototypes, drone and UAM workflows, and airport operations dashboards.

## Main Entry Points

- `notam_crawler_api.py`
- `notam_hybrid_crawler.py`
- `notam_monitor.py`
- `notam_change_detector.py`
- `reference_api/main.py`

## Quick Start

```bash
python3 -m venv .venv
source .venv/bin/activate
python3 -m pip install -r requirements.txt
python3 -m uvicorn reference_api.main:app --reload
```

Open `http://127.0.0.1:8000/docs` or query `http://127.0.0.1:8000/api/v1/notams`.

## Notes

- this is not an official government API
- upstream source formats may change
- verify against official sources for operational or safety-critical use
