# Use Cases

## Research And Education

- aviation safety analysis
- Korean NOTAM interpretation training
- airport and airspace change monitoring

## Product Prototypes

- flight-planning tools
- drone and UAM briefing products
- airport ops dashboards
- NOTAM alerting or summarization systems

## Infrastructure Value

This project is more infrastructure than end-user app.

- The direct value is not raw star count.
- The downstream value is enabling tools that need normalized Korean NOTAM data.
- A managed hosted API can sit on top of the same open-source core.
